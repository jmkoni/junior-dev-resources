Ruby

Ruby is an easy to learn, dynamic scripting language. It's a perfect choice if you are looking to get into web development.

More »

Python

Python is another easy to learn, dynamic scripting language. It's a great choice if you have a background in academia, sciences, or finance.

More »

Java

The oldest language we'll be looking at, Java is a strictly typed language and a bit more of a challenge to learn. However, it's what you want to learn if you are looking at getting a job with a more established company.

More »

JavaScript

Javascript is another dynamic language, but slightly harder to learn than Python or Ruby. However, it is one of the core technologies of the web. If you are slightly more interested in design, this language is your top choice.
